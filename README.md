# DOMJudge with React

## 개요

* React를 이용한 DOMJudge front-end 개발
* DOMJudge api에서 부족한 기능들과 로그인 등의 구현을 위해 express를 통한 api 서버 구축

## 설치

```bash
npm i && cd client && npm i && cd ..

npm start
```

## ESLint 확인

```bash
npm run lint

cd client
npm run lint
```
